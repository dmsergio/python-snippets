# Run tests from command line

### Unittest
```
python -m unittest test_calc  # run tests inside test_calc.py file
python -m unittest test_calc.py  # run tests inside test_calc.py file
python -m unittest -v test_calc  # run tests with verbose
python -m unittest discover  # run all test files found in the project
python -m unittest discover -v  # run all test files found in the project with verbose
```
